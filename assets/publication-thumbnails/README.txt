Add publication thumbnail images here.

For each publication in research.md, you can add a thumbnail image:

1. Create/find a 300x300px image (JPG format)
2. Save it here with a descriptive name
   Examples:
   - consensus-equilibrium.jpg
   - pnp-priors.jpg
   - face-id.jpg
   
3. Update the thumbnail path in research.md:
   <div class="publication-thumbnail">
     <img src="/assets/publication-thumbnails/your-image.jpg" alt="Paper title">
   </div>

Until you add images, thumbnails will appear as dark gray placeholders.

Image sources:
- Screenshot from your paper figures
- Visual abstract if you have one
- Representative result/diagram
- Create simple graphics in PowerPoint/Keynote
